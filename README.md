# Workout Timer 🏋️

音声付きでインターバルトレーニングができる  
シンプルな筋トレ用 Web タイマーです。

スマホでの利用を想定して作っています。

---

## 🔗 デモ（公開ページ）
https://sour39.github.io/workout-timer/

※ ブラウザでそのまま使えます  
※ Android Chrome で動作確認済み

---

## ✨ 特徴
- 音声でカウント・種目名を案内
- インターバルトレーニング対応
- スマホ操作に最適化したシンプルUI
- インストール不要（Webアプリ）

---

## 🏃‍♀️ 想定用途
- 筋トレ
- 体幹トレーニング
- インターバルトレーニング
- 自宅トレーニングの時間管理

---

## 🛠 使用技術
- HTML

- GitHub Pages

---

## 📱 PWAについて
PWA（ホーム画面に追加）にも対応していますが、  
Android Chrome の仕様やキャッシュの影響により  
URLバーが表示される場合があります。

Webアプリとしての利用は問題なく可能です。
